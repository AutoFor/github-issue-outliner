# GitHub Issue Outliner

GitHub の Issue を、親子関係つきで見やすく整理するためのアプリです。

このツールを使うと、以下のことができます。

- Issue を一覧で読みやすく表示する
- 親 Issue と子 Issue のつながりを階層で管理する
- 並び替えや編集を通して、作業計画を組み立てる

開発者向けの細かい設定を意識しなくても、Issue の全体像を把握しやすくすることを目的にしています。
