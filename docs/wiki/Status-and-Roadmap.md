# 現状と次アクション

このページは、プロジェクトの現在地を短く把握するための概要です。

## 今できること

- GitHub にログインして対象リポジトリを選択できる
- Issue をツリー（親子構造）で表示できる
- ドラッグ＆ドロップで並び替え・階層変更ができる
- サブ Issue 関連データを取得・更新できる
- キーボードショートカットで一部操作ができる

## ここまで作ったこと

- Next.js + TypeScript でアプリ基盤を構築
- Outliner の MVP（表示・並び替え・階層操作）を実装
- 開発モード（`GITHUB_TOKEN` を使ったローカル検証）を追加
- Wiki 基盤を初期化
  - `docs/wiki/` のページ作成
  - 図（draw.io + SVG）追加
  - `wiki-sync.yml` による自動同期設定

## これからやるべきこと（優先順）

1. Lint warning の解消（保守性の改善）
2. GitHub API 層の `any` を具体型へ置換（型安全性の改善）
3. GitHub 連携まわりのテスト追加（回帰防止）
4. 開発モードの安全ガード強化（本番誤設定の防止）
5. README を実運用向けに更新（導入手順・制約・既知課題）

## メモ

- Wiki は `main` ブランチへの `docs/wiki/**` 更新で同期される
- 図の更新は `docs/wiki/images/*.drawio` を編集して SVG を再エクスポートする
